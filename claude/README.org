#+TITLE: Claude - AI Assistant Configuration

* Overview

This charm provides global instructions and guidelines for Claude AI
assistant interactions with codebases. It establishes conventions for
commits, documentation, and development workflows.

* Structure

#+begin_example
claude/
├── src/
│   ├── CLAUDE.md              # Main guidelines (→ ~/.claude/CLAUDE.md)
│   └── references/
│       ├── commit-conventions.md
│       ├── git-push-convention.md
│       ├── new-project.md
│       ├── permission-levels.md
│       ├── agents-md-convention.md
│       └── kt-parse.md
└── metadata.yml
#+end_example

* Deployment

Files are linked to =~/.claude/=:

| Source           | Destination          |
|------------------+----------------------|
| =src/CLAUDE.md=  | =~/.claude/CLAUDE.md= |
| =src/references/= | =~/.claude/references/= |
